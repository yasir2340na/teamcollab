# 🛠️ TeamCollab

**TeamCollab** is a full-stack collaboration platform designed to help software teams manage their projects and tasks more effectively. It includes AI-assisted task suggestions and Kanban-style task organization.

---

## 🚀 Features

- ✨ Create and manage projects
- ✅ Add tasks to ToDo, Doing, and Done columns (Kanban board)
- 💬 AI-generated task suggestions using Cohere API
- 🧠 Backend API built with Express.js
- ⚛️ Frontend built with React.js (Vite)
